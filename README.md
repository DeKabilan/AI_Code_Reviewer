# AI_Code_Reviewer

Feature's:
1) Custom Rules and Standards
2) Rules Can be Enabled or Disabled
3) Can add Rule Exceptions
4) Review's Whole File Instead of MR Diff.

It can be Integrated into your Code Base Via,

1) Merge Request: On Submitting an MR you can add the applications endpoint as a Webhook Link to Trigger a Review
2) Using MR Link: If you have a Release Process, you can integrate this Tool into your Release Process, by Triggering the MR Endpoint with the MR Link to do a Review.
